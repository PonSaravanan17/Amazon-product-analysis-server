# 🛒 Amazon Product Analysis Server

An AI-powered product analysis platform that extracts and analyzes Amazon product information, including pricing, specifications, seller details, customer reviews, delivery information, and key features. Built with TypeScript and React components, the project provides structured insights for better purchasing decisions.

---

## 📌 Overview

Amazon Product Analysis Server is designed to fetch and organize product-related data into an interactive interface. It helps users understand product specifications, compare prices, analyze customer reviews, and view seller and delivery details in a clean and structured manner.

---

## ✨ Features

- 🔍 Product information extraction and analysis
- 💰 Price and deal tracking
- ⭐ Customer reviews and ratings analysis
- 📦 Delivery information display
- 🛍️ Seller details and reliability insights
- 📋 Product specifications and feature highlights

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm
- Bright Data API Key

### Installation

#### Clone the repository

```bash
git clone https://github.com/PonSaravanan17/amazon-product-analysis-server.git
```

#### Navigate to the project directory

```bash
cd amazon-product-analysis-server
```

#### Install dependencies

```bash
npm install
```

#### Configure environment variables

Create a `.env` file and add:

```env
BRIGHTDATA_API_KEY=your_api_key
OPENAI_API_KEY=your_openai_key
```

#### Start the server

```bash
npm start
```
